Waarp R66-Xml Config

Every command require an XML configuration file as first argument. This file contains some information that are really relative to the host. Those informations are described shortly in the following pages.

 

  • R66 Server Config
  • R66 Client Config

  • R66 Limit Config (to update the bandwidth limits dynamically)
  • R66 Rule Config (Rules Config file are the same including a root XML element named "rules" with the same subxml withinfor each rule)
  • R66 Authentication Config
