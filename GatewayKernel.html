Gateway Kernel

The Waarp Gateway Kernel module includes several packages that are needed by Gateway FTP and Gateway HTTP, and partially needed by others.

  • Database log on transfers
  • Http kernel methods for simple web page (manageable through XML and 2 Business Java classes)
  • Http REST kernel methods (used by Waarp R66): efficient simple REST implementation, with more than 5000 request/s on a 4-core x86 server.
