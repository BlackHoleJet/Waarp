Waarp Commons

We present here the "Commons" modules in Waarp project.

 

  1. Waarp Digest: This library takes into account MD5 in an optimized way.
  2. Waarp Common: Main common library shared with all projects in GoldenGate.
  3. Waarp LocalExec Daemon: used to execute on a simple external JVM the external command
  4. Waarp XmlEditor: based on XML Editor from F Golubov
  5. Waarp Password: used to generate DES key and passwords in GGP format
  6. Waarp SNMP: used to implement the SNMP Agent support for all Waarp project
  7. Waarp Gateway Kernel: used by Gateway FTP and Gateway HTTP for shared methods (database logs, actions, http shared package)

  8. Waarp Thrift: used by Waarp R66 to provide a Thrift interface for other languages than Java
